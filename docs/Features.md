# Features

## User Features
- Register with face capture (camera/upload)
- Face-based login
- Duplicate detection during signup
- Profile management

## Admin Features
- Secure admin login
- View all users
- View duplicate alerts
- Delete/block users
- Analytics dashboard

## AI Features
- Face detection (RetinaFace)
- Embedding generation (ArcFace)
- Cosine similarity matching
- Threshold tuning

## SaaS Features
- JWT authentication
- Role-based access control
- REST API architecture
- Scalable backend

## UX Features
- Real-time feedback
- Loading states
- Error handling
